# API Documentation

## Base URL
- Development: `http://192.168.1.24:8000`

## Endpoints
1. **Video Feed**
   - Endpoint: `/video_feed`
   - Method: `GET`
   - Usage: `<img src="/video_feed" alt="Video Feed">`

2. **Toggle Play/Pause**
   - Endpoint: `/ui/playpause`
   - Method: `POST`
   - Request: `{"action": "play" or "pause"}`
   - Response: `{"playing": boolean}`

3. **Toggle Snapshot**
   - Endpoint: `/settings/snapshot`
   - Method: `POST`
   - Request: `{"enable": boolean}`
   - Response: `{"snapshot_enabled": boolean}`

4. **Toggle Record**
   - Endpoint: `/settings/record`
   - Method: `POST`
   - Request: `{"enable": boolean}`
   - Response: `{"record_enabled": boolean}`

5. **Get Camera Status**
   - Endpoint: `/api/camera_status`
   - Method: `GET`
   - Response: `{"playing": boolean, "latest_snapshot_id": int, "latest_video_id": int}`

6. **Download Snapshot**
   - Endpoint: `/download/snapshot/{snapshot_id}`
   - Method: `GET`
   - Usage: `window.location.href = "/download/snapshot/1"`

7. **Download Video**
   - Endpoint: `/download/video/{video_id}`
   - Method: `GET`
   - Usage: `window.location.href = "/download/video/1"`
